# Currículo de Uma Página

Este projeto consiste no desenvolvimento de um currículo online de página única, focado em semântica HTML e boas práticas de estilização com CSS. O objetivo principal foi criar uma interface limpa, profissional e totalmente responsiva.

## Como Executar o Projeto

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` em qualquer navegador web moderno.
3. O currículo será exibido diretamente no navegador.

Alternativamente, se você tiver um servidor local instalado (como `live-server` ou `http-server`), execute o comando no diretório do projeto:

```bash
npx live-server
```

Ou simplesmente abra o arquivo `index.html` no navegador.
